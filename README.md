1. **Clone the repository**

   ```sh
   git clone https://github.com/odesai840/Space-Sim.git
   cd Space-Sim
   ```

2. **Run the setup script**
   
   ```sh
   cd Scripts
   .\Setup-Windows.bat
   ```
